09:00 - 12:30 mlr3 + drake

Author: Jakob Richter (TU Dortmund), Patrick Schratz (LMU Munich)

The workshop will be split into two parts.

The first part will introduce the new mlr3 package framework. mlr3 is the successor of the mlr package. We'll give a overview and walk you through some basic examples.

The second part starts with a brief overview of the drake package in R. This package helps you to set up a reproducible workflow of your project. It integrates greatly with the mlr3 package. We'll give examples on different complexity levels and walk you through these.
